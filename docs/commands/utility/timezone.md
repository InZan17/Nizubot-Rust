# Timezone Command
This command is used to check the time on different timezones.

Command is available on the user app.


## `/timezone set <timezone>`
**Parameters**

- `timezone` — Your timezone.

Sets your timezone.


## `/timezone remove`
Removes your timezone.


## `/timezone user <user?>`
**Parameters**

- `user` — User to check the timezone for. (Default: you)

Gets your/someones timezone.


## `/timezone check <timezone>`
**Parameters**

- `timezone` — The timezone to check the time for.

Gets the time at a specific timezone.
