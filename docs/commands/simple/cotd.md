# COTD (Color Of The Day) Command.

## `/cotd`
| Parameter | Description |
| :-------: | :---------- |
| `day`?    | Which day you wanna check the color for. (Default: Today) |

Gives you the color of the day. The color is randomly decided every day. It works by generating a random color and then picking the closest color from [this project](https://github.com/meodai/color-names).
